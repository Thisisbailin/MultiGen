# Verify — AIGC Create Module

- 选择分镜：在 Create 视图的顶部工具栏依次选择剧集与镜头，确认“提示词预览”区域实时更新字段摘要。
- 标签切换：点击“人物”与“场景”标签，Inspector 字段会依据模板（角色/场景）切换，Chip 状态正确。
- 文本生成流程：填写模板字段后点击“生成文本”，结果堆栈新增包含 Prompt 片段与模型名称的卡片，确认 `textService` 请求携带当前镜头与模板字段。
- 图像生成流程：点击“生成图像”后，堆栈新增包含图片缩略图的卡片；再次点击“设为 Base Image”并重新生成时，payload 会附带 base64/URL，堆栈标记来源。
- 模型/线路标签：顶部徽章随 Settings 中模型/线路切换而变化，确认 Create 视图继承当前线路信息。
- Base Image 导入：通过“Base Image”菜单导入本地 PNG/JPEG，生成时请求 payload 包含 `baseImageURL` 字段。
- 溶图流程：切换到“溶图创作”标签，分别选择角色图/场景图，点击“生成图像”后输出卡片并可再次设为 Base Image；抓包确认请求体同时包含角色/场景的参考图数据。
